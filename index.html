DisasterWatch

Monitor disasters areas with satellite imagery

Info

DisasterWatch is a crowdsourced plateform for event localisation and monitoring via remote sensed data.

Landsat-8 and Sentinel-2 image search results is powered by Developmentseed new Satellite API. Sentinel-1 image search is done via ESA scihub API.

Landsat and Sentinel images previews are from USGS Earth Explorer and Sentinel on AWS S3 (via Sinergise).

Continue reading: Blog post on RemotePixel.ca

HowTo

  • Check the events database ()
  • Use the to add an event or search for imagery
  • Use button to display MODIS/VIIRS daily/worldwide basemap
  • Latest EarthQuakes or Natural Events EONET available in

MODIS Terra: 2012/05/08 -> Today

MODIS Aqua: 2012/05/08 -> Today

MODIS instruments (Moderate-Resolution Imaging Spectroradiometer) on-board Terra and Aqua satellites provide a global view of earth on a daily basis since 1999 and 2002. With a resolution up to 250m per pixel and 36 spectral bands, MODIS images can produce stunning view of earth.

Only data produced after May 2012 are available on-line.

Late 2015 (Nov. 25), VIIRS imagery from SUOMI NPP spacecraft was also added to available global and daily dataset.

Data are available 3-4 hours after acquisition. if the map appears black, it means the data are not yet available.

Terra satellite local equatorial crossing time is approximately 10:30 a.m. in a descending orbit.

Aqua satellite local equatorial crossing time is approximately 1:30 p.m. in an ascending orbit.

SUOMI NPP satellite local equatorial crossing time is approximately 1:30 p.m. in an ascending orbit.
